Day 7 - Love Mondays!

07 September 2015

New week - new project.
This week we are supposed to continue improving our skills in OOP (last week's task and weekend challenge were also about using OOP).

OOP is object oriented programming. As it comes from the title, object is the main thing here. Same as in real life. And as in real life objects can communicate with each other.
To make an object first you need to make a class (it is like a blueprint, from which you can create objects). You can write a class, like this

class Dog

  attr_reader :name  

  def initialize (name)
    @name = name
  end

  def bark
    "Woof!"
  end

end

Then you can make a real object from this class!

my_dog = Dog.new('Shokky!')

And use this object to do stuff:

puts my_dog.name  
puts my_dog.bark

The result will be:

Shokky!  
Woof!

Easy, right?

So using this simple technic this week we are supposed to make Battle Ships game! And this time the instructions are minimal. We are working on our own!

Such a great field for creativity!

What classes should there be? how they should communicate with each other? who knows more - a ship or a board? do we need a board at all? are there any missiles involved? what are the rules? should we play this game first for an hour or so for the inspiration?

Love it!

Harry arranged a cake and we sang a birthday song! That was great! Best moment of the day.
