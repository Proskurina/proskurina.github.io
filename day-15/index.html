Today we learned a lot about communication between a client(mainly a browser) and a server, how a client(browser) sends request, a server receives it, executes some code and sends a response back to the client. Servers have to do the hardest work, I will try to write something about it later this week.

Also when we write a program that server should execute on request from a client, we use ruby, html and css, and we should keep them in separate files, and it seems now as a rather tricky part, maybe will also write something about it later.

p.s. Capybara is unbelievably cool!
